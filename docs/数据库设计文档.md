# 数据库设计文档

## 概述

本系统使用 PostgreSQL 作为主数据库，用于存储结构化的军事动态数据。

## 表结构

详见 `src/db/migrations/001_initial_schema.sql`

### countries（国家/地区表）
- `country_id` (UUID, PK)
- `code` (VARCHAR(3), ISO 3166-1 alpha-3)
- `name_zh` (VARCHAR(100))
- `name_en` (VARCHAR(100))
- `region` (VARCHAR(50))
- `created_at`, `updated_at`

### military_branches（军种表）
- `branch_id` (UUID, PK)
- `country_id` (FK → countries)
- `code`, `name_zh`, `name_en`, `branch_type`
- `created_at`, `updated_at`

### units（军事单位表）
- `unit_id` (UUID, PK)
- `country_id` (FK → countries)
- `branch_id` (FK → military_branches)
- `parent_unit_id` (FK → units, 自引用)
- `unit_code`, `unit_name`, `unit_level`, `unit_type`
- `is_active`, `established_date`, `dissolved_date`
- `created_at`, `updated_at`

### equipment（装备性能参数表）
- `equipment_id` (UUID, PK)
- `country_id` (FK → countries)
- `unit_id` (FK → units)
- `equipment_type`, `equipment_name`, `model_code`, `manufacturer`
- `specifications` (JSONB)
- `operational_status`, `inventory_count`, `deployment_location`
- `created_at`, `updated_at`

### positions（位置轨迹表）
- `position_id` (UUID, PK)
- `unit_id` (FK → units)
- `equipment_id` (FK → equipment)
- `position_type`, `latitude`, `longitude`, `altitude_m`, `accuracy_m`
- `position_source`, `reported_at`, `created_at`

### intelligence（情报记录表）
- `intel_id` (UUID, PK)
- `country_id` (FK → countries)
- `intel_type`, `title`, `content`
- `source_reliability`, `credibility`
- `location_description`, `latitude`, `longitude`
- `event_date`, `obtained_date`, `created_at`, `updated_at`

### alerts（预警记录表）
- `alert_id` (UUID, PK)
- `country_id` (FK → countries)
- `alert_level`, `alert_type`, `title`, `description`
- `source_intel_id` (FK → intelligence)
- `latitude`, `longitude`, `affected_area`
- `start_time`, `end_time`, `is_active`
- `created_at`, `updated_at`

### settings（系统设置表）
- `setting_id` (UUID, PK)
- `setting_key` (VARCHAR(100), UNIQUE)
- `setting_value` (TEXT)
- `setting_type`, `description`, `is_public`
- `created_at`, `updated_at`

## 索引

- `idx_units_country`, `idx_units_branch`, `idx_units_parent`
- `idx_equipment_country`, `idx_equipment_unit`
- `idx_positions_unit`, `idx_positions_equipment`, `idx_positions_reported`
- `idx_intel_country`, `idx_intel_type`, `idx_intel_event_date`
- `idx_alerts_country`, `idx_alerts_level`, `idx_alerts_active`
