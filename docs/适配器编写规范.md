# 适配器编写规范

## 概述

适配器（Adapter）是爬虫模块的核心组件，每个信息源对应一个适配器类。所有适配器继承自 `BaseAdapter`。

## 适配器接口

```python
from src.crawler.base import BaseAdapter, ParseResult

class MyAdapter(BaseAdapter):
    def __init__(self):
        super().__init__("my_adapter_name")

```

## ParseResult 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `unit_id` | str | 单位标识符 |
| `event_type` | str | 事件类型（movement/exercise/deployment/incident） |
| `title` | str | 情报标题 |
| `content` | str | 情报内容 |
| `source_url` | str | 来源 URL |
| `source_type` | str | 来源类型 |
| `confidence` | float | 置信度（0.0 ~ 1.0） |
| `latitude` | float | 纬度 |
| `longitude` | float | 经度 |
| `event_date` | datetime | 事件时间 |

## 编写新适配器

1. 在 `src/crawler/adapters/` 目录下创建新文件
2. 继承 `BaseAdapter`
3. 实现 `fetch()` 方法
4. 使用 `ConfidenceCalculator` 计算置信度
5. 返回 `ParseResult` 列表
