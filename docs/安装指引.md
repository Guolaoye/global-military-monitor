# 安装指引

## 环境要求

- Python 3.10+
- PostgreSQL 15+
- Deepin Linux / Windows / macOS

## 步骤

### 1. 克隆项目

```bash
git clone <repository_url>
cd global-military-monitor
```

### 2. 安装依赖

**Linux:**
```bash
pip install -r requirements.txt --user --break-system-packages
```

**Windows:**
```bat
pip install -r requirements.txt
```

### 3. 配置环境变量

复制 `.env.example` 为 `.env`，填写实际配置：

```bash
cp .env.example .env
```

### 4. 初始化数据库

```bash
psql -U postgres -c "CREATE DATABASE global_military_monitor;"
psql -U postgres -d global_military_monitor -f src/db/migrations/001_initial_schema.sql
```

### 5. 配置 Obsidian Vault

在 Obsidian 中打开 `~/Documents/军事知识库/` 作为保险库。

### 6. 启动应用

```bash
python src/main.py
```

## 验证安装

```bash
python src/db/connection.py
```

输出 `数据库连接成功` 即表示安装成功。
