# 用户手册

## 概述

全球军事动态分析研判系统是一个基于 AI 的军事情报分析平台。

## 功能模块

### 1. 情报收集

系统自动从以下来源收集情报：
- AIS 船舶追踪数据
- 各国国防部官方公告
- 军事自媒体（微博、微信公众号等）
- 民间军事数据站

抓取频率可在设置中调整（默认每2小时一次）。

### 2. 地图展示

- 高德地图 / 天地图双底图切换
- 军事图标叠加显示（陆军、海军、空军等）
- 图标聚合与展开
- 历史轨迹回溯

### 3. 力量结构

- 树状图：国家 → 军种 → 部队层级结构
- 网状图：单位之间的关系可视化

### 4. 态势分析

- 真实态势：历史事件时间轴
- 模拟推演：复制单位进行参数调整推演
- 毁伤计算：基于兵棋引擎的战斗力评估

### 5. 预警系统

当系统检测到重大军事动态时，通过飞书机器人推送预警通知。

## 设置

在设置界面可以配置：
- 数据库连接
- 高德/天地图 API Key
- 飞书机器人 App ID / Secret
- AI 模型 API Key
- 爬虫抓取频率

## 常见问题

**Q: 地图无法加载？**
A: 检查 API Key 是否配置正确，或者切换到备选底图。

**Q: 情报抓取失败？**
A: 检查网络连接，或者调整爬虫间隔。

**Q: 数据库连接失败？**
A: 检查 PostgreSQL 是否启动，配置是否正确。
